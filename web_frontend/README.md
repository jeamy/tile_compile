# GUI2 Web Frontend

Produktives HTML-Frontend fuer das FastAPI-Backend.

Wichtig:
- Startseite ist das Dashboard (`/ui/` -> `index.html`).
- Keine separaten Navigationsseiten fuer Layout-Maße oder Workflow-Uebersichten.
- Fokus auf produktive Screens und reale Bedienpfade.

## Start

1. Backend starten:

```bash
cd /media/data/programming/tile_compile
./start_backend.sh
```

2. Frontend im Browser:

- `http://127.0.0.1:8080/ui/`
